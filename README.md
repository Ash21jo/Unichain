# Unichain
A group effort, it focused on building a shared blockchain network for handling overseas tuition transfers. Think of it as a digital pipeline meant to simplify how students send school payments across borders.
The big aim? Show that blockchains built without code can tackle high costs, slow processing times, often stretching days, along with swings in exchange rates. These issues hit hard - especially learners from lower-income regions sending money to universities abroad. What stood out was watching simple tools cut through traditional banking friction.
